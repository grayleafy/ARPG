# ARPG
Unity3D ARPG小游戏 实现了对话、任务、背包、武器系统，可以存档读档，有简短的剧情，行为树控制角色AI，IK控制脚贴地，AnimationRigging控制持枪瞄准和武器切换
学习Unity后的第一个探索性质的ARPG游戏，大部分系统都简单的实现了但是没有追求细节，后期觉得写得不够好然后摆烂准备下一个项目
